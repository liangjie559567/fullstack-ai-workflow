# Release Notes - v0.1.0

## Summary
First publishable release of the Fullstack AI Workflow template repository.

## Includes
- Claude Code / Cursor / Codex template entrypoints
- Manifest-driven workflow repository installation
- Bootstrap / dispatch / template apply scripts
- CLAUDE.md / AGENTS.md / testing instructions / PRD / SLICE templates
- GitHub-ready repository files
- Validation workflow and release drafter setup

## Recommended Adoption Steps
1. Create a repository from this template or fork it
2. Replace placeholder mirror URLs in `workflow-repos.manifest.example.json`
3. Publish tag `v0.1.0`
4. In downstream projects, pin `ref` to `v0.1.0`
