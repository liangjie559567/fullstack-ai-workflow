# Prompt Snippets

## Bootstrap Workflow
Initialize repository workflow templates without overwriting existing files.
Create missing files, map commands to current stack, and produce a summary.

## Explain Current Stage
Read CLAUDE.md, STATE.md, CONTEXT.md, and current slice.
Then explain which stage the project is in and what should happen next.

## Prepare Slice Execution
Read the active SLICE.
Output: pre-slice checklist, Red/Green/Refactor checklist, files likely to change, and tests to add.
