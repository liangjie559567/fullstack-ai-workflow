# Universal Fullstack AI Workflow

## Stage 1 - Discuss
Clarify goal, constraints, scope, and acceptance.

## Stage 2 - Plan
List files, services, commands, risks, and test strategy.

## Stage 3 - Slice
Break work into independently verifiable vertical slices.

## Stage 4 - Execute
Write tests first when possible, then implement minimal code.

## Stage 5 - Verify
Run lint, typecheck, tests, build, and e2e as needed.

## Stage 6 - Ship
Summarize change, evidence, risk, rollback, and observability items.
